Test Aggregation Mode

This page tests the aggregation mode implementation by loading the ver3/index.html in an iframe.

Test Case: Load example RDF and switch to Aggregation mode

  1. Click "Turtle" to load example data
  2. Select "Режим агрегации" from the dropdown
  3. Click "Визуализировать"
  4. Verify that:
    • Literals (like "John Doe", "30") are shown inside the node boxes
    • Name/label predicates have normal font size
    • Other literal predicates have smaller font size
    • rdf:type and rdfs:subClassOf predicates are hidden by default
